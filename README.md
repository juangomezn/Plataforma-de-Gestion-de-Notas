# 🎓 Acme School - Plataforma de Gestión de Notas

**Acme School** es una plataforma web en desarrollo orientada a la gestión académica de colegios. Su objetivo es facilitar el registro de usuarios, la administración de perfiles y en etapas posteriores, la gestión de notas, cursos y reportes académicos.

---

## ⚙️ Tecnologías utilizadas

### Frontend
- React JS (Vite)
- React Router DOM
- CSS personalizado

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- Passport (OAuth2)
- Express Validator

---

## 🚧 Estado del proyecto

Este proyecto se encuentra en desarrollo. Actualmente incluye:
- Formulario de perfil de usuario con validaciones
- Página de bienvenida
- Navbar responsive (en proceso de mejora)
- Selección de ciudad con código y nombre

Próximas funcionalidades incluyen autenticación completa, gestión de notas y generación de reportes.
